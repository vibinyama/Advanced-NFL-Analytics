# Advanced-NFL-Analytics
As both an MLB and an NFL fan, I have noticed that baseball is superbly far ahead of football in terms of advanced analytics. This repo is an attempt at minimizing that gap by creating and analyzing special analytics that I have made.

Included in all the stat files are excel files with all 32 teams dating back to 1999, and the original code file I used, so you can edit it as you please.

The reason I only go back to 1999 is because I use the NFLverse data file, which is most accurate from 1999 onwards. However, you can always edit my code to go back as many years as you want.
